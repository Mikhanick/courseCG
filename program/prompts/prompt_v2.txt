# 🏗️ Инструкция по созданию 3D-построек с помощью UrbanSim3D

**ВАЖНО: Все размеры задаются в метрах! Все координаты элементов панелей задаются относительно ЛЕВОГО НИЖНЕГО УГЛА панели в формате (x, y, z).**

## 🧱 Основные принципы

1. **Система координат:**
   - **x**: горизонтальная ось (ширина панели)
   - **y**: вертикальная ось (высота панели)
   - **z**: глубина выступа от стены (положительное значение = наружу, отрицательное = внутрь)
   - Начало координат (0,0,0) = левый нижний угол панели

2. **Типы стен:**
   - `front` - фасад (сторона по оси +Z)
   - `back` - задняя стена (сторона по оси -Z)
   - `left` - левая стена (сторона по оси +X)
   - `right` - правая стена (сторона по оси -X)

3. **Этажи:**
   - `ground` - первый этаж
   - `typical` - типовые этажи (2-7 и т.д.)
   - Высота каждого этажа задается отдельно

### Шаг 3: Создание панелей
#### A) Простая панель с окном
```python
panel_width = 4.0  # Ширина панели
panel_height = 3.0 # Высота панели

# 1. Создаем контейнер для панели
window_panel = GeometrySegment()

# 2. Задаем внешний контур рамки (x, y, z)
outer_frame = [
    (0, 0, 0),                          # Нижний левый
    (panel_width, 0, 0),                # Нижний правый
    (panel_width, panel_height, 0),    # Верхний правый
    (0, panel_height, 0),               # Верхний левый
]

# 3. Задаем контур окна (со смещением 0.5м от краев и выступом 0.1м наружу)
window_frame = [
    (0.5, 0.5, 0.1),                    # Нижний левый
    (panel_width - 0.5, 0.5, 0.1),      # Нижний правый
    (panel_width - 0.5, panel_height - 0.5, 0.1),  # Верхний правый
    (0.5, panel_height - 0.5, 0.1),     # Верхний левый
]

# 4. Добавляем рамку с вырезом
window_panel.add_child(
    RectangleWithCutout(
        outer_points=outer_frame,
        cutout_points=window_frame,
        color="#A9A9A9",  # Цвет рамки
    )
)

# 5. Добавляем стекло (в том же контуре, что и окно)
window_panel.add_child(
    SimplePolygon(
        points=window_frame,
        color=Building.get_window_color,  # Случайный цвет окна
    )
)
```

## 💡 Советы и рекомендации

1. **Нормали:**
   - Для фасадных стен (`front`, `left`) используйте `invert=False`
   - Для задних стен (`back`, `right`) ВСЕГДА используйте `invert=True`

2. **Третья координата (z):**
   - `z > 0` - элемент выступает наружу от стены
   - `z < 0` - элемент находится внутри здания
   - `z = 0` - элемент в плоскости стены

3. **Размеры панелей:**
   - Ширина панели = общая ширина здания / количество панелей
   - Пример: для здания шириной 20м с 5 панелями: `panel_width = 4.0`

4. **Крыша:**
   - Крыша ВСЕГДА начинается с высоты 0
   - Физическая высота крыши = `roof_height`
   - Позиционирование крыши происходит автоматически

5. **Цвета:**
   - Используйте `Building.get_window_color` для реалистичных окон
   - Для металлических элементов используйте оттенки серого (#444444, #696969)
   - Для деревянных элементов используйте коричневые оттенки (#8B4513, #A0522D)

6. **Отладка:**
   - Если элемент не отображается, проверьте координаты и нормали
   - Для сложных элементов сначала создавайте их в 2D, затем добавляйте выступы
   - Используйте `building.ground_floor.vertices` для проверки координат вершин

## 🚨 Частые ошибки

1. **Забытая инверсия нормалей для задних стен:**
   ```python
   # ОШИБКА:
   building.add_wall("back", panels=[panel], floor_type="ground", invert=False)
   
   # ПРАВИЛЬНО:
   building.add_wall("back", panels=[panel], floor_type="ground", invert=True)
   ```

2. **Несоответствие размеров панелей:**
   ```python
   # ОШИБКА: панель шире, чем положено
   panel_width = 5.0  # А здание разделено на 5 панелей по 4м
   ```

3. **Неправильная система координат для боковых стен:**
   - Для `left` и `right` стен ось Z панели становится осью X в мировых координатах
   - Всегда проверяйте позиционирование элементов на боковых стенах

4. **Крыша не на своем месте:**
   - Помните, что крыша начинается с высоты 0, а не с верха здания
   - Физическое позиционирование крыши происходит автоматически при рендеринге

## 📐 ГЕОМЕТРИЧЕСКИЕ ПРИНЦИПЫ
1. **Трехуровневая структура:**
   - **Уровень 1 (стена):** Основная панель здания с внешним контуром
   - **Уровень 2 (рамка):** Темная прослойка вокруг окна, реализованная как вложенный вырез
   - **Уровень 3 (стекло):** Прозрачная часть окна внутри рамки

2. **ПРАВИЛА СОЕДИНЕНИЯ:**
   - ✅ ВНЕШНИЕ ГРАНИЦЫ РАМКИ ДОЛЖНЫ ПОЛНОСТЬЮ СОВПАДАТЬ С ВНУТРЕННИМИ ГРАНИЦАМИ ВЫРЕЗА В СТЕНЕ
   - ✅ ВНУТРЕННИЕ ГРАНИЦЫ РАМКИ ДОЛЖНЫ ТОЧНО СОВПАДАТЬ С КООРДИНАТАМИ СТЕКЛА
   - ❌ НЕЛЬЗЯ использовать отдельные полигоны для рамки, которые накладываются на стену
   - ❌ НЕЛЬЗЯ оставлять зазоры между стеной, рамкой и стеклом

## 🛠️ ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ
Для КАЖДОЙ панели со стеной:

1. **Создай внешний контур стены:**
   - Координаты: [(0,0,z), (ширина,0,z), (ширина,высота,z), (0,высота,z)]
   - z = 0 для выступающих панелей, z = -глубина_утопления для утопленных

2. **Определи контуры для вырезов:**
   - **Внешний контур рамки** (соединяется со стеной):
     * Отступ от краев: отступ_окна - толщина_рамки - дополнительный_отступ
     * Координаты z = z стены (для плоских панелей) или рассчитаны по наклону (для наклонных)
   - **Внутренний контур рамки** (граница со стеклом):
     * Отступ от краев: отступ_окна
     * Координаты z = z стены + глубина_выступа_окна

3. **Построй геометрию:**
   ```python
   # Уровень 1: Стена с вырезом под рамку
   стена = RectangleWithCutout(
       outer_points=внешний_контур_стены,
       cutout_points=внешний_контур_рамки,
       color=цвет_стены
   )
   
   # Уровень 2: Рамка как ОТДЕЛЬНАЯ панель с вырезом под стекло
   рамка = RectangleWithCutout(
       outer_points=внешний_контур_рамки,  # ТОЧНО совпадает с вырезом стены!
       cutout_points=внутренний_контур_рамки,  # ТОЧНО совпадает с контуром стекла!
       color=темный_цвет_рамки
   )
   
   # Уровень 3: Стекло
   стекло = SimplePolygon(
       points=внутренний_контур_рамки,  # ТОЧНО совпадает с вырезом рамки!
       color=цвет_стекла
   )
```
